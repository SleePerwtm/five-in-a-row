## 仓库说明

该仓库为嵌入式系统课程的实验作业之一：使用 QT 开发的五子棋游戏，在 Linux 系统上编译和运行。

## 实验要求：使用 QT 编写五子棋程序

使用QT设计一个图形化五子棋游戏程序。 该程序应该具备如下基本功能：

1. 棋盘图形界面

   创建比较友好的人机交互界面。

2. 轮流对弈

   黑棋先行，两人轮流对弈，可以通过程序自动判断胜负。

3. 悔棋功能

   可以悔棋至上一步。

4. 棋局保存功能

   可以把每一个棋局的步骤保存下来，可以使用文件保存，也可以通过数据库保存。

5. 棋局的复盘功能

   可以加载已经保存的棋局，通过前进与后退，进行复盘。

6. 附加功能
   
   可以设置黑棋禁手，该功能为可选功能。

3. 详细说明程序设计步骤；
